安全之路

在上大学前没用过电脑 对电脑一窍不通 连开机都不会 鬼知道我这一年经历了什么 大一国庆的时候才买了人生第一个笔记本 刚到手的时候连开机都踉跄 不过如今已经很熟练

大一上 军训完就开始琢磨我的个人选择 毕竟计算机有很多个方向 大部分能接触最多的就是码代码 我花了半个月去学了html和css 发现这和我想象的计算机不一样 我觉得很无聊 所以开始研究计算机的各个方向 比如 软件开发 网络工程 嵌入式 算法 等

软件开发(纯码代码太无聊了)

网络工程(装网线的 纯xx专业 不要问我为什么)

嵌入式(当时听说要物理好 遂放弃)

算法(一堆刁钻的数学问题 我懒)

我的专业是网络工程 我更喜欢调侃成给别人装网线的 事实上也确定有装网线的 我不喜欢这么专业 大一的课程都是思科华为网络配置什么的 极其无聊 以至于我差点就挂科 是真的无聊 现在还在后悔当时应该趁早转的 因为每当期末都要花时间去学 纯纯浪费我的时间

我极其讨厌算法 其实更多讨厌的是各种刁钻的数学问题 什么时间复杂度 最优算法 看得我脑壳痛 至于为什么当时没选择嵌入式是因为听别人说要对电路什么的了解 我物理极差 遂放弃 最后选择的是网络安全 一开始被吸引是因为黑客两个字 帅 入行后才发现很无聊

以上所说其实只是在大一国庆的时候 也就是说我大一开学一个月后就找到了自己喜欢的方向(其实没那么喜欢)

以下是我的安全学习经历

大一上

11-1月

我把html css js 都完了 紧接着把b站上那个计算机入门科普视频看完(这里很推荐入门看这个 很有用) 接着就开始真正学习安全 常见的漏洞 例如sql注入 rce命令执行 我一开始选择搭建了dvwa 因为学习sql注入我就跑去学了sql 那时候因为不是很懂电脑 每天最多浪费的时候就是搭建各种环境 记得第一次搭建靶场时我连续坐在电脑前疯狂google问题 那时候还不知道gpt的存在 很多资料甚至是国外的 只能硬着头皮看

1-3月

打完dvwa后就开始对安全有了一定的了解 后面又开始看小迪的视频 两倍速连续十来天直接看完了 后开始搭建靶场vuluhub 开始有了正规的训练 让我对整个渗透流程有了很强的认识 直接把小迪看得东西消化了 从信息收集 web渗透 内网渗透 提权 因为需要内网渗透 我又学了linux 编写shell脚本 后为了方便安装了kali 为了更完善的工具集 那段时间 每天两眼一睁就是打靶场 这是我进步神速的时候 我并没有很好的基础 所以那个时候就是照着wp打 不懂就上外网看资料 那是我最痛苦的时候 因为啥也不懂 基础也没打好 直接上手 跟不带枪上战场没啥区别 不过还是硬着头皮啃下来了 鬼知道我是怎么坚持下来的

3-4月

开始发现自己的code能力太弱 于是觉得学点开发 当然不是软件工程那种 而是囫囵吞枣的过了一遍 php shell python c java go c cpp csharp

开始接触ctf 一开始打的是网上的一些ctf平台 说实话很乱 不够系统 后面开始在ctfshow有规模的打 打了两个月赶紧溜 具体原因说来话长 当然这是我做的非常对的一个选择 得亏跑得快 而后转行去挖src 挖了一个星期获得人生第一个漏洞(虽然是公益) 而后在一个星期内挖了一堆漏洞 随后还是跑路了 因为我觉得这些挖洞的 一半以上做的事情实际就是软件测试 当时我认为这与我想的hacker不一样 第一次知道短信轰炸也算漏洞的我也算是震惊了….没绷住

4-6月

开始打hackthebox和tryhackme(极其推荐) 他和前面说的vulnhub一样 但是htb更难 打了两个月后水平神速提升 对整个渗透流程有了很深刻的理解 从信息收集到web渗透再到内网的横向到提权 可以说各种奇奇怪怪的攻击手法都用过了 那时候第一次感觉到知道和真实利用其实是两种状态 后又刷了小迪和暗月的视频 主要是巩固 到这里其实我个人觉得基本算入行了

6-7月

因为打hackthebox经常接触内网 于是把win和linux的内网都给学了一遍 内网信息收集 到横向 提权 维权各种手法

也开始接触域环境 开始系统学习域环境 当时几乎是把域的手法都学了一遍了 什么kerberos攻击 各种认证攻击 中间人攻击 pth ptt票据注入 以及黄金,白银,蓝宝石各种票据维权手法,域的常用nday,spn攻击,组策略攻击,ldap和winrpc的各种利用,滥用dc,域的三种委派攻击,域信任攻击和域证书攻击 14种 我记得当时是有14种还是几种来着 域的攻击手段特别多而且特别复杂 正如你看到的 写这篇博客的时候 已经全忘了…….老实说只要几天不接触全忘没了 而且域的工具命令特别复杂 需要考虑各种情况

7-8月

开始研究代码审计 主要是java安全。因为之前打ctf经常练php反序列化 也有了一定的基础 开始打java安全的靶场和审计靶场的代码。但是因为没有开发经验很难受 遂放弃

8-9月

开始学习免杀技术 在选择语言的时候做了很多的纠结 主要的用来免杀的语言有python go rust c cpp ps1脚本 最后选择了cpp 因为万金油 代码审计一样 因为没有开发能力和对操作系统内核的了解以及受迫于国内并没有好的免杀教学 遂 寄

9-10月

决定硬磕代码审计 主要我真不想学开发 后面还是克服了 跟着文章分析cc1-cc10的链子还有什么jdbc链子 反正一堆链子的反序列化 已经忘完了 什么ldap,rmi各种服务的利用啊 什么xxx版本的jdk滥用啊 xxx链子构造链滥用 出网和不出网的利用 以及一些著名框架漏洞什么的 log4j weblogic springboot等 后面还有内存马的 那三种类型 什么滥用xx版本tomcat打链子的 甚至研究到tomcat底层原理实现 一开始看确实头疼 后面理解觉得 发现这些发明原创攻击手法的人简直是天才 感慨code能力弱真的寸步难行

10-11月

相对安卓小程序攻防了解 所以开始学习安卓逆向 当然只是很简单的部分 例如hook 工具脱壳反编译 反hook抓包 分析函数 逆向一些数据包字段等 不过最后还是只会用自吐算法的脚本了….cnm frida的hook不知道忘哪去了 脱壳也是一点都不会…..得亏逆向的大佬工具都很吊 让我在安卓逆向也可以实现ida一把梭….没绷住…….. 同时学了一点点js逆向 主要是逆下登录算法

11-12月

开始学习一些简易的win逆向 一开始是因为觉得能破解东西什么的很帅 后面才发现有多坐牢 而且学习东西一堆 我跟的是海哥的基础课和中级班(强烈推荐) 逆向这部分其实很水 但主要我不是专门搞逆向的 倒也无所谓 主要对了操作系统已经内核有了很好的见解 什么时候才能啃懂csapp(*꒦ິ⌓꒦ີ) 后面了解了一点win的内核 window的内存管理句柄分页模式进线程异常处理等

1月-2月

开始着手之前放弃的免杀 有了之前学逆向看的操作系统 pe结构 以及内核 基本算是轻车熟路 花了一周的时候就成功绕过了主流的杀软 包括卡巴 某60和某绒和defender 第一次绕过某60的时候给我激动坏了 因为之前听说国内某60很强 下面的图是vt的 没全过(猎鹰真的强)

实际的经历比这复杂很多 只是打字好累不想打了(呜呜呜X﹏X)

这辈子最幸运的是我在新手入学后一个月就找到自己的方向 并且我本身也算喜欢(可能)()->(现在没那么喜欢……)

2.19二更

如果对安全真的有兴趣 建议先学半年开发先 后续可以做安全研究或者安全开发 以前不懂这个 后面才发现就算再强 也只是个水平高一点的工具侠 实际目前安全圈几乎全是这样的人 如果你只是想当个安服 那倒是无所谓 如果真的想成为红队 就不能只是局限于web渗透

其实英语对于安全真的很重要 我目前还是上大二 我对自己的要求是毕业前词汇量达到8k以上 当你水平一直在提高 实际后面都是在阅读纯英文章(老外技术是真的强) 一定要学会纯英阅读 这个是真的很重要

最近都在疯狂卷开发 本来当时就是讨厌开发才走的安全 气死我了

2月-3月

继续卷开发 学了点go 目前代码审计的go项目越来越多了 最近开始着手web3安全 了解了一些web3基本知识 现在在学sol开发

最近好累 学安全真的巨多知识

web渗透

内网渗透

域安全

代码审计

免杀

安卓小程序攻防

js逆向

云安全

这上面随便一个拿出来都够累了 工资又比不过开发 真心劝退不是热爱赶紧转行

最近在学点web3安全 了解了一些sol编程。hardcat和foundry框架和top10web3漏洞

最近

web3暂时放下了 知识量太大了 影响我的web2安全 估摸着等毕业再开始着手

这段时间都在学习开发 学学代码审计 主要是go java php以及几个框架tp springboot gin等 也算了挖到一些比较水的0day漏洞 虽然都没啥用

最近开发能力提升了不少 于是想到刚好会点免杀 于是索性免杀处理了一下fscan 说实话挺水的 不过最后还是过了数字某绒

还研究了一点点勒索 主要是文件加密这块 感觉还是需要免杀功底 这几天在研究webshell工具 主要是哥斯拉和冰蝎

-–

续更

这段时间水了几个cve 免杀处理了几个安全工具 fscan和哥斯拉 目前在处理mimikatz 总的来说还是符合预期 不过要达到专业的redteam 仍还有很长的路去走 接触的英语资料越来越多 也需要考虑提升下英语水平了 后面想考个雅思

5月-8月

这段时间是我进步飞速的时间 因为打好了很多基础 最近都是在做安全研究 7月入职了国内某安全厂 研究一些病毒样本 学到了很多恶意开发的知识！

最近在尝试写一个c2？嗯 不是开发出生确实很难 但也不是为了做出一个项目 毕竟市面上成熟的产品太多了 主要是理解一下c2的整个架构！

后面9月还要回去上学！从社畜再次转向学生

9月

开始准备上学的事情咯 这几天一直在写c2 完成大概有40%???maybe md c怎么什么都没有…

还有半年就要结束学校生活 当初许诺的学好英语早已抛掷脑后…呜呜呜

单词量还需要一点点提升 虽然大多数有关安全单词的都会学会了 但在一些复杂的安全博客 议会等还是有不少晦涩的单词

听力真的太弱了 水平能停留在初中水平 目标到达2000小时 2025前完成200小时!因为现在一直着手学校和安全的事情 英语都没怎么练了

10月

9月的时候写了第一份c2框架 但是由于实在不熟悉go 而且屎山代码太多了 于是废除了这版 后面重新写了一版 目前v1版已经写完了 大体上c2该有的功能也都有了 但肯定还是比不过商业化的c2

励志成为红队第一糕手的第n天!
